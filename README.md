# Chatbot Flow Builder

A minimal React Flow Builder for creating chatbot conversation flows.

## Features

- Canvas-based flow builder using React Flow
- Drag-and-drop Message nodes
- Text node editor in settings panel
- One outgoing edge per node rule
- Flow validation on save

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
